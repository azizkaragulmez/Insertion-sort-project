data structures 1st projectt

Proje 1:

Insertion Sort Aşamaları:

[22, 27, 16, 2, 18, 6] (Başlangıç durumu)
[22, 27, 16, 2, 18, 6] (22 sabit, sonraki elemanın uygun yerini bulmaya çalışır)
[16, 22, 27, 2, 18, 6] (16 uygun yere yerleştirilir)
[2, 16, 22, 27, 18, 6] (2 uygun yere yerleştirilir)
[2, 16, 18, 22, 27, 6] (18 uygun yere yerleştirilir)
[2, 6, 16, 18, 22, 27] (6 uygun yere yerleştirilir)
Big-O Gösterimi: O(n^2)

18 Sayısının Time Complexity Durumu:
18 sayısı, dizi sıralandıktan sonra Average Case kapsamına girer. Çünkü aradığımız sayının ortada olma ihtimali en yüksektir.
